# README

## Installation

### Composer

```sh
composer require webblu/phpmyhelper 
```

### Manually

Download, Extract and `require/include` the src file at where needed.

```php
require_once __DIR__ . '/../src/Helper.php';

use WebbLu\Helper;

// codes...
```
